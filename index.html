Case Study
Restaurant Online Ordering Live Admin

A complete restaurant platform, end to end.

A full ordering system for a Karachi nihari restaurant — from customer cart to live admin dashboard to kitchen print slips. Built to replace Foodpanda commission with direct ownership.

  • Cinematic dark editorial design with Urdu calligraphy and custom typography
  • Filterable menu with 24+ items across 7 categories, real-time cart
  • Six Karachi delivery zones with custom fees, minimum order detection
  • Live Firebase backend — orders save instantly to a real database
  • Admin dashboard with real-time order sync, status updates, audio alerts
  • One-click kitchen receipt printer with thermal-printer-friendly layout
  • Auto-WhatsApp customer when order is ready
